# Strata Trace Plugin

Execution tracking and persistence forensics plugin for Strata.

Strata Trace identifies evidence of program execution, persistence mechanisms,
and defense evasion techniques across forensic images. It detects LOLBINs,
Prefetch artifacts, Scheduled Tasks, AmCache entries, BAM/DAM timestamps,
Registry Run key persistence, BITS job abuse, and timestomping indicators.

Each detection is mapped to the relevant MITRE ATT&CK technique and assigned
a forensic significance level to support triage and reporting workflows.
