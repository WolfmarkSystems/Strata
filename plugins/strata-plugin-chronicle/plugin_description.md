# Chronicle Plugin

Chronicle reconstructs user activity timelines from forensic evidence sources. It parses Prefetch execution records, browser history databases, LNK shortcut files, Windows Event Logs, RecentDocs MRU entries, TypedPaths navigation history, WordWheelQuery search terms, Windows Timeline (ActivitiesCache.db), and CapabilityAccessManager consent records to build a comprehensive picture of user behavior on a system.
