## Reporting a Vulnerability

Strata processes forensic evidence in high-stakes environments. Security issues are taken seriously.

**Do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities directly to Wolfmark Systems:

- **GitHub:** Open a private security advisory at [github.com/WolfmarkSystems/Strata/security/advisories](https://github.com/WolfmarkSystems/Strata/security/advisories)
- **Email:** security@wolfmarksystems.com *(coming soon — use GitHub advisory in the meantime)*

### Response Timeline

- **Acknowledgment:** within 72 hours
- **Assessment:** within 7 days
- **Fix target:** based on severity — critical issues prioritized
