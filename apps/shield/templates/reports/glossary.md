# Glossary

| Term | Definition |
|------|------------|
| MFT | Master File Table - NTFS filesystem metadata structure |
| IOC | Indicator of Compromise - evidence of malicious activity |
| Carving | Recovery of files from raw disk data |
| FTS | Full-Text Search |
| NSRL | National Software Reference Library |
| Chain of Custody | Documented history of evidence handling |
| Hash Chain | Tamper-evident log using cryptographic hashes |

## Acronyms

- SHA: Secure Hash Algorithm
- MD5: Message-Digest Algorithm 5
- UTC: Coordinated Universal Time
- DB: Database
- SQL: Structured Query Language
