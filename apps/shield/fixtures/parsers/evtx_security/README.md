# EVTX Security Fixtures

This fixture set provides 30 synthetic, labeled Security event samples for CLI and parser validation.

- `win10/`: UTF-8 XML fragments stored with `.evtx` extension (15)
- `win11/`: UTF-16LE XML fragments stored with `.xml` extension (15)
- `manifest.json`: generated index of fixture metadata and expected event IDs

These fixtures are synthetic and intended for deterministic parser tests and contract checks.
