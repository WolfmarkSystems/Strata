# EVTX Sysmon Fixtures

This fixture set provides 30 synthetic Sysmon event samples for deterministic parser/CLI validation.

- `win10/`: UTF-8 XML fragments stored with `.evtx` extension (15)
- `win11/`: UTF-16LE XML fragments stored with `.xml` extension (15)
- `manifest.json`: generated fixture metadata and event-id labels

These fixtures are synthetic and intended for repeatable tests and contracts.
