# UX Acceptance Checklist (Phase 3)

1. Analyst can load evidence table and apply text/type/category filters.
2. Analyst can switch from table to timeline without blocking UI.
3. Analyst can save and reload timeline filters.
4. Analyst can pivot by source facet in timeline.
5. Analyst can inspect row details in preview panel.
6. Analyst can export selected rows.
7. Analyst can open hex preview for selected row.
8. Analyst can open strings preview for selected row.
9. Analyst can see parse completeness status (parsed/partial/unsupported).
10. Analyst can navigate >50k rows with virtualization enabled.
