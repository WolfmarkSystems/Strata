# Windows Large-Image Stress Gate Runbook

Purpose: capture repeatable runtime and memory metrics for large-image CLI runs without changing engine/backend architecture.

## Run

From repo root:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File scripts/windows_large_image_stress_gate.ps1 -ImagePath D:\forensic-suite\evidence\large_image.E01
```

Optional tuning:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File scripts/windows_large_image_stress_gate.ps1 -ImagePath D:\forensic-suite\evidence\large_image.E01 -MftCount 10000
```

## Output

Results are written under:

`_run/windows_roadmap/stress/<utc_stamp>/`

Artifacts:

1. `stress_gate_summary.json`
2. `stdout.txt`
3. `stderr.txt`
4. `smoke_out/smoke_summary.json` (when generated by smoke-test)

Recorded metrics include:

1. `elapsed_ms`
2. `peak_working_set_mb`
3. `peak_private_bytes_mb`
4. `exit_code`
5. smoke summary status (if available)
