# STRATA KNOWLEDGE: FORENSIC EVIDENCE DATASETS (META-ARCHIVE)

This guide documents the world’s most significant forensic datasets, providing Strata with the ground-truth metadata required for cross-validation and tool testing.

---

## 📧 THE ENRON EMAIL CORPUS (~500,000 Emails)
- **Origin**: 2001 investigation into the Enron Corporation.
- **Forensic Value**: The gold standard for social network analysis (SNA), keyword searching, and communication pattern discovery.
- **Strata Insight**: High-density link analysis of executive communication logs.

---

## 🏛️ NIST CFReDS (COMPUTER FORENSIC REFERENCE DATA SETS)
- **DD-DiskImages**: Standardized raw images for file recovery testing.
- **Metadata**: [NIST CFReDS Portal](https://www.cfreds.nist.gov/)
- **Core Sets**:
    - **Leaked Documents**: 10,000 documents with varied metadata for search testing.
    - **Malware Images**: Infected VM snapshots for memory and system forensics.
    - **NSRL (National Software Reference Library)**: 100M+ hashes of known software (RDS).

---

## 🇺🇸 THE CLINTON EMAIL DATABASE (~30,000 Emails)
- **Forensic Value**: Use of private servers for government business.
- **Strata Insight**: Analysis of SMTP headers and X-Headers for routing persistence.

---

## 🏆 THE DFRWS CHALLENGES (2001-2024)
- **2001 (Intrusion)**: The first grand challenge in the field.
- **2010 (Smartphones)**: Early Android/iOS forensic baseline.
- **2023 (Cloud)**: AWS/S3 exfiltration and persistence.

---

## 🧬 SYNTHETIC DATA GENERATORS
- **G-DATA**: Tools for generating realistic, timestamped user activity in virtual environments.
- **ForensicPy**: Scripts for injecting artifacts (Registry, LNK, Cache) into clean images for validation.
