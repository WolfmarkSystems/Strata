# Local AI & LLM Tools for Enhancement

## Self-Hosted LLM Platforms

### Ollama
- **Stars**: 157K
- **GitHub**: https://github.com/ollama/ollama
- **Description**: Easy setup for running Llama, Mistral, Gemma locally
- **Use case**: Could replace or supplement llama-server

### LocalAI
- **Stars**: 42K
- **GitHub**: https://github.com/mudler/LocalAI
- **Description**: Drop-in OpenAI replacement, no GPU required
- **Features**: Text, Audio, Video, Images, Voice Cloning

### Open WebUI
- **Stars**: 116K
- **GitHub**: https://github.com/open-webui/open-webui
- **Description**: User-friendly chat interface for local LLMs
- **Use case**: Alternative UI to current Wolf Sentinel chat

### GPT4All
- **Stars**: 76K
- **GitHub**: https://github.com/nomic-ai/gpt4all
- **Description**: Run LLMs locally on any device
