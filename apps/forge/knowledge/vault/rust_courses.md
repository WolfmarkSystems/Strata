# Free Rust Programming Courses & Resources

## Comprehensive Free Courses

### 1. Comprehensive Rust (Google)
- URL: https://google.github.io/comprehensive-rust/
- Developed by Android team at Google
- Covers: Full spectrum from basic syntax to advanced topics (generics, error handling)
- Available in: English, Portuguese, Chinese, Japanese, Korean, Farsi, Spanish, Ukrainian
- Also available as 2-day intensive: https://comp-rust-2day.rust-edu.org/

### 2. Rust From Zero To Hero (Rust Z2H)
- URL: https://rustz2h.com/
- Interactive online book
- Project-based learning
- What you'll build:
  - Safe and Concurrent Systems Software
  - Blazingly Fast Command-Line Tools
  - High-performance Web Services with Actix/Rocket

### 3. 100 Exercises To Learn Rust
- URL: https://rust-exercises.com/100-exercises/
- Hands-on exercises
- Teaches core concepts, syntax, type system, standard library, ecosystem
- Assumes knowledge of at least one programming language

### 4. Rust Programming Essentials (Coursera)
- URL: https://coursera.org/learn/rust-programming-essentials
- Part of Programming with Rust Specialization
- 4 modules, ~1 week to complete
- Beginner level

### 5. Rust Tutorial for Beginners (Medium)
- URL: https://medium.com/@a1guy/rust-crash-course-for-beginners-2025-learn-rust-programming-from-scratch-and-build-a-102ad751e730
- Crash course series
- Build WebAssembly Ready Apps

### 6. Master Rust Programming - LearningRust.org
- URL: https://learningrust.org/
- Structured beginner-friendly lessons
- Hands-on examples and practical exercises
- Try Rust in 30 seconds (online playground)

### 7. The Rust Book (Official)
- URL: https://doc.rust-lang.org/book/
- The official documentation
- Most recommended resource for learning Rust

### 8. Rust by Example
- URL: https://doc.rust-lang.org/rust-by-example/
- Learn Rust through examples

## Key Rust Concepts to Master

1. Ownership and Borrowing
2. Lifetimes
3. Pattern Matching
4. Error Handling (Result, Option)
5. Concurrency (async/await, tokio)
6. Generics and Traits
7. Cargo and Crates.io
8. Unsafe Rust
9. FFI (Foreign Function Interface)
10. Testing and Benchmarking

## Recommended Tools

- rustup - Rust toolchain manager
- cargo - Package manager and build tool
- rust-analyzer - Language server
- Clippy - Linter
- Rustfmt - Code formatter
- Ruffle - WASM playground

## Best Practices

- Use `rustfmt` for consistent code style
- Run `cargo clippy` for linting suggestions
- Write tests with `#[test]` attribute
- Use documentation comments `///` for public APIs
- Prefer safe Rust, use unsafe only when necessary
- Leverage the type system for correctness
