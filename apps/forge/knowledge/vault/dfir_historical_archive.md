# STRATA KNOWLEDGE: HISTORICAL FORENSIC ARCHIVE (1990-2025)

This archive preserves the foundational research that shaped modern digital forensics, providing Strata Chat with context on how artifacts have evolved over 35 years.

---

## 🕰️ THE FOUNDATIONAL ERA (1990 - 2000)
- **1991: The Sleuth Kit (TSK) Foundations**: First implementations of file system analysis tools for Unix.
- **1998: DFRWS Born**: The Digital Forensic Research Workshop establishes the first academic standards for the field.
- **1999: SANS Reading Room established**: Centralized repository for "hands-on" forensic research.

---

## 🚀 THE METADATA REVOLUTION (2001 - 2010)
- **2002: Prefetch Introduced (Windows XP)**: Microsoft adds prefetching to speed up boots; forensic examiners realize it's a perfect execution trace.
- **2005: Volatility 1.0**: Memory forensics moves from niche research to industry requirement.
- **2007: iPhone Launch**: Mobile forensics begins the shift from simple bit-copies to complex database parsing (SQLite).
- **2008: Registry Research Peak**: Deep understanding of the Winreg hive structure (M Harlan, et al.).

---

## 🛡️ THE MODERN ERA (2011 - 2024)
- **2014: Unified Logging (Apples)**: macOS and iOS move to a binary log format (`.tracev3`), breaking traditional text-string analysis.
- **2019: Windows Timeline**: Windows 10/11 begins recording user actions across devices in `ActivitiesCache.db`.
- **2021: Cloud Forensics Dominance**: Shift toward API-driven artifact collection from AWS/Azure/GCP.
- **2024: The AI Frontier (Windows Recall)**: Operating systems begin taking snapshots of everything the user sees, creating the ultimate forensic artifact.
