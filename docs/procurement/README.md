# Strata Forensics — Procurement Package

This directory contains procurement and evaluation materials for Strata Forensics. These documents are intended for government procurement officers, agency technical evaluators, and legal counsel — not software developers.

## How to Use This Package

1. **Start with the Product Brief** to understand what Strata Forensics does and how it fits your agency's needs.
2. **Review the Compliance Matrix** to verify alignment with your procurement requirements and security policies.
3. **Use the Technical Evaluation Guide** during hands-on evaluation to independently verify every claim made in the other documents.
