# Strata Tree User Manual

## Start
1. Launch `strata-tree.exe`.
2. Set examiner profile.
3. Create or open a `.vtp` case.

## Core Workflow
1. Open evidence (E01/RAW/directory).
2. Wait for indexing to complete.
3. Review tree, table, preview, and hex tabs.
4. Run `HASH ALL` and load hash sets.
5. Bookmark notable file/registry artifacts.
6. Export PDF/CSV/HTML reports.

## Tabs
- `FILE EXPLORER`: file tree/table/preview.
- `REGISTRY`: hive tree, values, quick keys, search.
- `TIMELINE`: chronological events + suspicious filters.
- `GALLERY`: image thumbnails + EXIF/GPS warning.
- `COMPARE`: evidence A/B diff.
- `EVENT LOGS`: EVTX high-value/suspicious events.
- `BROWSER HISTORY`: visits/downloads + suspicious markers.
- `HASH SETS`: known-good/known-bad import and matching.
- `AUDIT LOG`: chained entries + integrity status.

## Forensic Safety
- Evidence sources are read-only.
- All timestamps are UTC.
- Every key action is audit logged.
