# Artifact Reference

## Prefetch
Execution metadata for programs (`.pf`). Use for run counts and run times.

## Shellbags
Folder navigation history from user hives (`NTUSER.DAT`, `UsrClass.dat`).

## LNK
Shortcut target metadata, useful even if target is deleted.

## Browser History
Visited URLs, titles, transitions, and downloads from Chrome/Edge/Firefox DBs.

## EVTX
Windows Event Logs; prioritize IDs 4624/4625/4688/7045/1102.

## Registry Hives
SYSTEM/SOFTWARE/SAM/SECURITY/DEFAULT/NTUSER.DAT for config and persistence.
