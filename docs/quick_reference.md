# Examiner Quick Reference

- Open Case: `OPEN CASE`
- Open Evidence: `+ OPEN EVIDENCE`
- Hashing: `HASH ALL`
- Carving: `CARVE`
- Report: `REPORT`
- Bundle Export: `EXPORT`

## Filters
- Deleted only: set file filter to `$deleted`
- Known bad only: set file filter to `knownbad`

## Chain of Custody
- Check `AUDIT LOG` tab banner: `Chain integrity: VERIFIED`
- Export audit to `CSV/JSON/PDF` for court package.
